# photomod
Bu proje, Raspberry Pi 4B  üzerinde çalışan bir fotoğraf kabini (photobooth) uygulamasıdır. Flask, Pygame ve Pillow gibi kütüphaneleri kullanır.

## Gereksinimler

- Python 3
- Raspberry Pi Camera (libcamera destekli)
- Flask
- Pygame
- Pillow
- libcamera
